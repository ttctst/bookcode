# 書籍線上檔案
這個檔案為書本編碼的補充檔案，你會在這裡，看到更為詳盡的編碼，以及一些補充的內容，此外書中錯誤的地方，也會透過相連的編碼，加上補充說明，你可直接在線上查看，亦可把壓縮檔案下載下來
